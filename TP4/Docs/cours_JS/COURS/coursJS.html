1. STRUCTURE DE BASE

Lorsque le code JavaScript se trouve dans un fichier séparé (.js), on suit cette structure :


/* zone d'en-tête */
/* déclaration des variables globales */
let declaration1;
/* définition des fonctions */
function maFonction() { ... }

2. LES COMMENTAIRES

Les commentaires permettent une relecture de programme (travail en équipe).

  • // : Commentaire sur une ligne
  • /* ... */ : Commentaire multi-lignes

3. DECLARATION DE VARIABLES

JavaScript est "faiblement typé". On utilise let pour une variable et const pour une constante.

let a = 3; // a est un Number

4. LES DIFFÉRENTS TYPES

Le JavaScript a 8 types principaux : Number, String, Boolean, Null, Undefined, BigInt, Object et Symbol.

5. FONCTIONS D'ENTRÉE/SORTIE

  • window.alert("Message"); : Affiche une boîte.
  • window.prompt("Question"); : Demande une saisie.
