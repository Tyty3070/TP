1. Structure de base d'un programme

Lorsque le code JavaScript se trouve dans un fichier séparé (.js), la structure est relativement simple[cite: 107]:

/* zone d'en-tête */
/* déclaration des variables globales */
/* corps du programme */ [cite: 108, 109]

Dans une page Web, on utilise la balise <script> entre <head> ou en fin de <body>[cite: 115].
