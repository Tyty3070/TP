1. Structure de base d'un programme

Lorsque le code se trouve dans un fichier séparé (.js), la structure est simple:

/* zone d'en-tête */
/* déclaration des variables globales */
/* corps du programme */

Dans une page Web, on utilise la balise <script> entre <head> ou en fin de <body>[cite: 113, 115].

2. Les Commentaires

Ils permettent une relecture facilitée du programme[cite: 116].

  • // pour une ligne[cite: 119].
  • /* ... */ pour plusieurs lignes[cite: 120].

3. Les Variables

JavaScript est "faiblement typé"[cite: 126]. On utilise let ou var[cite: 129, 150].

Types principaux : Number, String, Boolean, Null, Undefined[cite: 131].

4. Les Opérateurs

Arithmétiques (+, -, *, /, %, **) [cite: 187] et Logiques (&&, ||, !, ==, !=)[cite: 205, 206].
